# Game Premise

## One-sentence form

The Pit is a text-first narrative puzzle game in which the player enters a world of unfinished processes and must decide, under uncertainty, which paths are worth commitment and which only consume effort.

## Hidden thesis

The project is built on a concealed computability structure, but the fiction should never name it.

The experiential translation is:

> The player can become better locally, but can never become complete globally.

This matters because the game should generate a specific feeling:

- small situations yield real understanding
- larger systems reward pattern recognition
- the strongest heuristics eventually fail lawfully
- the player ends with judgment, not omniscience

## The player's real question

Do not foreground a sterile yes/no prediction question.

The player's internal question should feel like:

- Is this path worth committing to, or will it trap me?
- Does this resolve, or does it only consume?
- Am I making progress, or feeding a loop?
- Should I continue here, or act elsewhere?

That shift is essential. The game is about finite attention under uncertainty, not abstract theorem recital.

## Narrative promise

The world should feel lawful, not arbitrary.

Some mechanisms open.
Some rituals complete.
Some growths bloom.
Some corridors return in altered form.
Some relics absorb effort without closure.

The player should gradually infer that there is a deep order governing why some things conclude and others do not.

## Non-negotiable constraints

- Do not mention programs, Turing machines, halting, decidability, or proof language in the game fiction.
- Do not present the game as random, cruel, or anti-player.
- Do not withhold progress just to look mysterious.
- Do allow many real wins.
- Do ensure that delayed contradiction targets methods, not isolated guesses.
- Do keep the world story-rich and regionally grounded rather than abstract and sterile.

## Core interaction loop

The baseline experiential loop is:

`observe -> hypothesize -> commit -> receive partial consequence -> reassess`

The commitment step is what makes the game emotionally real. Without it, the game becomes detached analysis.

## Why the game is special when it works

The player is not solving one giant locked answer. The player is learning how to live and act in a world where some things can be solved, some can only be navigated, and the deepest skill is learning the difference.
